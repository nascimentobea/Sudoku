# Sudoku Solver in Prolog

## This project implements a Sudoku solver in Prolog that validates and solves both:

- Standard 9x9 Sudoku (with 3x3 subgrids).

- General NxN Sudoku (where N is a perfect square, e.g., 4x4, 9x9, 16x16).

### Features:

- Validates rows, columns, and sub-matrices.

- Handles incomplete boards (zeros are treated as empty cells).

- Works for both classic Sudoku and generalized NxN Sudoku.
